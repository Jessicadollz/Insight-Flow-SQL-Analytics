# Insight Flow SQL Analytics

## 📌 Problem Summary

This project focuses on analyzing customer transaction data using SQL and data analytics techniques. The objective is to extract meaningful insights, perform data cleaning, and apply analytical methods such as RFM (Recency, Frequency, Monetary) analysis to understand customer behavior and segmentation.

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/<your-username>/Insight-Flow-SQL-Analytics.git
cd Insight-Flow-SQL-Analytics
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run

* Open the notebook from the `notebooks/` folder in Google Colab or Jupyter.
* Run all cells sequentially to perform:

  * Data loading
  * Data cleaning
  * SQL-based analysis
  * RFM analysis
  * Visualization
